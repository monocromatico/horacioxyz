📷 imgopt.php – Optimizador de Imágenes

Este script permite redimensionar y optimizar imágenes locales o remotas mediante parámetros en la URL. Es ideal para generar versiones adaptadas de imágenes en tiempo real y servirlas cacheadas desde el directorio assets/.

🔧 Parámetros disponibles

  • src – Ruta de la imagen (relativa o URL remota)
  • max-width – Ancho máximo en píxeles
  • max-height – Alto máximo en píxeles
  • ratio – Proporción deseada (ej: 1:1, 16:9)
  • crop – Si se debe recortar para mantener la proporción (1 = sí, 0 = no)
  • quality – Calidad de compresión (1–100)
  • format – Formato de salida: jpg, png, webp

📁 Carpeta de caché

Las imágenes optimizadas se almacenan automáticamente en el directorio assets/ para evitar reprocesamiento innecesario.

🧩 Estructura por lotes

Opcionalmente, puedes organizar imágenes en carpetas individuales por nombre o fecha/hora, como:

/imageopt/
└── nombre-o-fecha/
    ├── uploads/           ← Imágenes originales
    ├── assets/            ← Optimización general (sin presets)
    └── assets/400x300/    ← Presets con dimensiones específicas

Esto permite gestionar proyectos o lotes de forma independiente.

🖼️ Ejemplos de uso

Reducir ancho a 400px

?src=sample.jpg&max-width=400

Reducir alto a 300px

?src=sample.jpg&max-height=300

Proporción 1:1 con recorte

?src=sample.jpg&max-width=300&ratio=1:1&crop=1

Formato WebP

?src=sample.jpg&format=webp

Imagen remota

?src=https://placehold.co/600x400/orange/white?text=Remota&max-width=400

📚 Cómo funciona el script

  1. Lee los parámetros de la URL.
  2. Genera un hash único para crear un nombre de archivo de caché.
  3. Verifica si el archivo optimizado ya existe en assets/ (o subcarpeta específica).
  4. Si no existe:
    • Descarga la imagen (si es remota).
    • Redimensiona según max-width / max-height.
    • Recorta si se especifica ratio y crop=1.
    • Guarda el archivo optimizado en assets/ o su subcarpeta correspondiente.
  5. Entrega la imagen optimizada con cabeceras correctas.

🧠 Ideal para proyectos web que necesiten versiones livianas de imágenes sin preprocesarlas manualmente.
